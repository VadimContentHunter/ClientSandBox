# ClientSandBox

ClientSandBox — это приложение для Windows, предназначенное для удобного запуска и управления **sing-box** через графический интерфейс.
> **Внимание:** ClientSandBox протестирован с **sing-box 1.13.14**. При использовании других версий рекомендуется предварительно проверить совместимость конфигурации и корректность работы приложения.

## Совместимость

Приложение разработано и протестировано с:

- **sing-box 1.13.14**

Работа с другими версиями sing-box не гарантируется.

---

# Возможности

- Запуск и остановка sing-box.
- Выбор исполняемого файла `sing-box.exe`.
- Выбор конфигурационного файла.
- Просмотр логов в реальном времени.
- Автоматический запуск через Планировщик заданий Windows.
- Работа из системного трея.
- Сохранение настроек между запусками.

---

# Системные требования

- Windows 10 x64 или Windows 11 x64
- .NET Runtime не требуется (Self-Contained сборка)
- Права администратора

---

# Установка

1. Запустите `ClientSandBox-Setup.exe`.
2. Следуйте инструкциям мастера установки.
3. После установки запустите **ClientSandBox** от имени администратора.

---

# Первоначальная настройка

При первом запуске необходимо указать:

- путь к `sing-box.exe`;
- путь к конфигурационному файлу (`config.json`).

После сохранения настроек их не потребуется вводить повторно.

---

# Использование

1. Запустите ClientSandBox.
2. Выберите нужный inbound.
3. Нажмите **Запустить**.
4. Для завершения работы нажмите **Остановить**.

Во время работы можно просматривать журнал событий во вкладке **Логи**.

---

# Автозапуск

ClientSandBox поддерживает автоматический запуск через **Планировщик заданий Windows**.

Использование Планировщика позволяет запускать приложение с повышенными правами администратора.

---

# Важно

ClientSandBox **не содержит** sing-box.

Перед использованием необходимо самостоятельно скачать совместимую версию:

- **sing-box 1.13.14**

и указать путь к `sing-box.exe` в настройках программы.
